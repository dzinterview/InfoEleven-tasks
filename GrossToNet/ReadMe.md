# Calculate NetPrice from GrossPrice

## Installation

```
npm install
```

## Run tests

```
npm test
```

### Options

| Properties       | Type   | Default    | Description                                                           |
| ---------------- | ------ | ---------- | --------------------------------------------------------------------- |
| grossPrice       | number | \*required | GrossPrice                                                            |
| VAT              | number | \*required | VAT in percent (e.g: 20.5)                                            |
